## 关于 Demo 版本说明

本仓库会持续新增 Demo，`main` 分支内容会持续变化。

但每个 Demo 目录在对应文章发布后，代码会保持"冻结状态"，确保读者从文章进入仓库时，能看到与文章完全匹配的代码实现。

后续功能演进和 API 更新会在新的 Demo 目录中继续展开。

---

## 如何使用

1. 根据文章找到对应的 Demo 目录
2. 进入该目录查看 README 了解运行方式
3. 获取 API Key：https://tickdb.ai
